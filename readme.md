# Personal Website

A simple personal website built using **HTML** and **CSS** as part of a beginner web development project.

## Features

* Responsive navigation bar
* Hero section with profile image
* About Me section
* Projects section
* Contact section
* Custom CSS styling
* Clean and modern layout

## Built With

* HTML5
* CSS3

## Project Structure

```
.
├── index.html
├── style.css
├── README.md
└── DEVLOG.md
```

## How to Run

1. Clone or download this repository.
2. Open `index.html` in your web browser.

No additional software or dependencies are required.

## Deployment

This website can be deployed using services such as:

* GitHub Pages
* Netlify
* Vercel
